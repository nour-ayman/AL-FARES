# AL-FARES

## Project Description
**AL-FARES** is a full-stack web application designed to manage and display **school and university supplies**.  
The project includes both **frontend** and **backend** components, following a **Feature-First Architecture** to make the code modular, maintainable, and easy to scale.  

The application is organized around the following **core features**:

1. **Auth** – User login, registration, and authentication.  
2. **Products** – Displaying products, product details, and related components.  
3. **Categories** – Organizing products by categories with filtering.  
4. **Search** – Searching products with real-time suggestions and results.  
5. **Cart** – Adding products to the shopping cart and managing quantities.  
6. **Checkout** – Completing orders, entering shipping info, and processing payments.  
7. **Home** – Homepage with featured products, banners, and category highlights.
